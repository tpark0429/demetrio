# Basic Definition
## Unit

## DB Scale
$B=log\frac{P_2}{P_1}$, 1 Bel = 10 deciBels (dB)
$dB=10log_{10}\frac{P_2}{P_1}$ : 본래 power의 비이나 전류나 전압의 비로 나타낼 땐 $20log_{10}\frac{V_2}{V_1}$ 형태가 된다. 
($\because P\propto V^2$)
    
if $P_1=P_2, dB=-dB$
    $P_1>P_2, dB>0$
    $P_1<P_2, dB<0$
if $P_2=2P_1, dB=10log_{10}\frac{2P_2}{P_1}=3dB$ (+3dB는 2배, -3dB는 1/2배)
    $P_2=4P_1 : 6dB$
    $dBm=10log_{10}\frac{P}{1mW}|_{600\Omega}$ : 통신 시스템에서 세운 기준으로 부하가 600 ohm 일때 1 mW 소모하는 것 기준
if  $P=40mW : 16dB$
    $dBm=10log_{10}\frac{40mW}{1mW}=10(1+0.6)=16dB$
    $P=4W : 36dB$
