# Parameter

매개변수, 파라미터, 모수는 수학과 통계학에서 어떠한 **시스템이나 함수의 특정한 성질**을 나타내는 변수를 말한다.

일반적으로 $\theta$ 라고 표현되며, 다른 표시는 각각 독특한 뜻을 지닌다.

함수의 값을 구하서나 시스템 반응을 결정할 때는 독립변수는 변하지만 매개변수는 일정하다.

- 독립변수 : 다른 변수에 영향을 미치는 변수
- 종속변수 : 독립변수의 영향을 받아 일정하게 변하는 변수
- $y=ax+b$ 에서 x는 독립변수, y는 종속변수, a와 b는 매개변수
- 파이썬 예시
```python

def my_sum(a: float, b: float) -> float:
    return a+b

my_sum(10, 20) : a와 b는 parameter, 10과 20은 argument (파라미터의 값으로 전달되는 것)
```

## 확률 분포에서의 파라미터

### 이항 분포

- $X~B(n,p)$
- 매개변수 n과 p가 주어졌을 때 n번 시행 중에 k번 성공할 확률

### 정규 분포

$X~N(\mu, \sigma^2)$

- 모수, 모평균, 모분산이 주어졌을 때 확률변수 X가 x일 확률
- 매개변수 : 평균($\mu$), 분산($\sigma^2$)

## Parameter와 Deep Learning

- 최종 출력 값이 정답과 가장 가깝게 만드는 parameter 조합을 찾는 과정
- Parameter 조합을 찾는 방법은 오차가 최소화 되도록 미분을 통해 모든 parameter를 한꺼번에 업데이트한다.
