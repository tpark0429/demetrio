# Parameter

매개변수, 파라미터, 모수는 수학과 통계학에서 어떠한 **시스템이나 함수의 특정한 성질**을 나타내는 변수를 말한다.

일반적으로 $\theta$ 라고 표현되며, 다른 표시는 각각 독특한 뜻을 지닌다.

함수의 값을 구하서나 시스템 반응을 결정할 때는 독립변수는 변하지만 매개변수는 일정하다.

- 독립변수 : 다른 변수에 영향을 미치는 변수
- 종속변수 : 독립변수의 영향을 받아 일정하게 변하는 변수
- $y=ax+b$ 에서 x는 독립변수, y는 종속변수, a와 b는 매개변수
- 파이썬 예시
```python

def my_sum(a: float, b: float) -> float:
    return a+b

my_sum(10, 20) : a와 b는 parameter, 10과 20은 argument (파라미터의 값으로 전달되는 것)
```

## 확률 분포에서의 파라미터

### 이항 분포

- $X~B(n,p)$
- 매개변수 n과 p가 주어졌을 때 n번 시행 중에 k번 성공할 확률

### 정규 분포

$X~N(\mu, \sigma^2)$

- 모수, 모평균, 모분산이 주어졌을 때 확률변수 X가 x일 확률
- 매개변수 : 평균($\mu$), 분산($\sigma^2$)

## Parameter와 Deep Learning

- 최종 출력 값이 정답과 가장 가깝게 만드는 parameter 조합을 찾는 과정
- Parameter 조합을 찾는 방법은 오차가 최소화 되도록 미분을 통해 모든 parameter를 한꺼번에 업데이트한다.

# Probability concepts
## Odds
- 실패와 성공의 비율을 설명하는 값 (실패 비율 대비 성공 비율로 정의)
- $$odds=\frac{p}{1-p}$$
  - p : probability of success

### Odds 값의 범위
- Success 횟수가 fail 보다 많을 떼 : (1,$\inf$)
- Success 횟수가 fail 보다 적을 떼 : (0,1)
- Success 횟수=fail 횟수 : 1

### Odds 문제점
- 비율을 적용하다 보니 1보다 클 경우의 scale과 1보다 작을 경우의 scale이 다르다.
- 모든 Odds 값에 대한 대칭적 해석이 필요 (직관적 이해 어려움)
- 따라서 확률의 공리 (특정 사건에 대한 모든 확률의 합은 1)를 만족 시키고 직관적 해석을 위해 odds에 log를 취한다.

## Logit (Log + Odds)
- Odss값에 밑을 자연상수e를 취한 것
- $logit(p)=\ln(\frac{p}{1-p})$
- Logit은 deep learning에서 수많은 hidden layer를 거쳐 출력된 일종의 확률 정보를 담고 있는 최종 판단을 위한 실수값 (Activation Function을 통과하면서 성공과 실패의 비율이 누적된 값)
- 해당 Neuron 값이 클수록 성공 확률이 높다는 의미는 담겨 있으나 숫자의 범위가 -inf ~ +inf 로 직관적 해석이 어렵다. 따라서 이를 확률값으로 변환이 필요
 - $L=ln(\frac{p}{1-p}) \to e^L = \frac{p}{1-p} \to p = \frac{e^L}{1+e^L} \to p = \frac{1}{1+e^{-L}}$
 - p : success probability
 - e : natural logarithm
 - L : Logit (output of hidden layer)
